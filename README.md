# 📌 사하구청 AI 상담사 프로토타입 개발

## 📖 1. 프로젝트 개요 및 배경 (PRD)
* **문제 정의:** 최근 공공기관 홈페이지는 다양한 행정 정보와 민원 안내 서비스를 제공하고 있으나, 메뉴 구조가 매우 복잡하여 사용자가 필요한 정보를 직관적으로 찾기 어려운 한계가 존재합니다. 
* **해결 방안 및 목표:** 사하구청 공식 홈페이지의 실제 행정 데이터를 기반으로, 사용자의 자연어 질의를 이해하고 필요한 정보를 즉시 제공하는 'AI 상담사 프로토타입'을 개발합니다. 비정형 행정 데이터를 구조화하고 하이브리드 검색 엔진을 도입하여, 주민들이 보다 편리하게 행정 서비스를 이용할 수 있는 지능형 환경을 구축하는 것을 목표로 합니다.
* **타겟 사용자:** 민원 신청 절차, 행정 서비스, 복지 정보 등 다양한 정보를 빠르고 편리하게 확인하고자 하는 **사하구 주민 및 홈페이지 방문자**.

## ⚙️ 2. 핵심 기능 및 기술 스택
* **비정형 데이터 구조화:** 홈페이지 크롤링 및 LLM 기반 메타데이터 자동 태깅
* **하이브리드 검색:** 메타데이터 필터링(1단계) + Vector 의미 검색(2단계)
* **멀티턴 대화:** 문맥 유지 및 역질문을 통한 사용자 의도 파악
* **기술 스택:** Python, LangChain, RAG, FastAPI, Vector DB(FAISS/Chroma)
