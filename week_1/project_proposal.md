## 2. 문제 정의
- [cite_start]최근 공공기관 홈페이지는 다양한 행정 정보와 민원 안내 서비스를 제공하고 있으나, 메뉴 구조가 복잡하여 사용자가 필요한 정보를 찾기 어려운 문제가 있습니다[cite: 22]. [cite_start]기존의 홈페이지 검색 방식은 접근성이 떨어지는 한계가 있어, 자연어 질의를 통해 즉시 정보를 제공하는 시스템이 필요합니다[cite: 23, 24].

## 3. 핵심 기능 3가지
1. [cite_start]**하이브리드 검색 시스템:** 메타데이터 기반 필터링(1단계)과 Vector 기반 의미 검색(2단계)을 결합하여 검색 정확도를 극대화합니다[cite: 55, 69, 72, 75].
2. [cite_start]**멀티턴 대화 처리 기능:** 사용자의 이전 대화 문맥을 유지하고, 질문이 모호할 경우 역질문을 통해 사용자의 의도를 분석하여 응답합니다[cite: 56, 78, 79, 80, 81].
3. [cite_start]**자연어 답변 및 출처 제공:** 검색된 행정 정보를 일반인이 이해하기 쉬운 자연어로 요약하고, 답변의 근거가 되는 홈페이지 URL 출처를 함께 제공합니다[cite: 57, 83, 84, 85].
